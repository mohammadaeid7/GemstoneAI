# Gemstone AI (Android)

تطبيق أندرويد (Jetpack Compose) للتعرّف على صور الأحجار/المجوهرات باستخدام الذكاء الاصطناعي على الجهاز (ML Kit Image Labeling)
ثم تقدير قيمة تقريبية بناءً على: **الوزن (قيراط) + الجودة + النقاء**.

## ملاحظات مهمة جداً
- نموذج ML Kit الافتراضي **عام** (General) وليس مُدرّباً خصيصاً على كل أنواع الأحجار الكريمة. لذلك الدقة الواقعية لتحديد نوع الحجر قد تكون محدودة.
- تقدير السعر في هذا المشروع **افتراضي/تعليمي** (Offline) وليس سعراً سوقياً رسمياً. للتقييم الحقيقي يلزم فحص مختبري وشهادة.

## تشغيل محلياً
1) افتح المشروع في Android Studio
2) Run على جهاز/محاكي

## الحصول على APK مجاناً عبر GitHub Actions (بدون كمبيوتر قوي)
1) أنشئ حساب GitHub (مجاني)
2) أنشئ Repository جديد وارفع هذا المشروع كاملاً
3) من تبويب **Actions** شغّل Workflow باسم **Build Android APK**
4) بعد اكتمال البناء ستجد ملف APK في **Artifacts** باسم: `app-debug-apk`

الـ APK الناتج هو Debug APK. يمكن لاحقاً تجهيز Release Signing إذا احتجت نشره على المتجر.

## أين تعدّل جدول الأسعار؟
داخل الملف:
`app/src/main/java/com/example/gemstoneai/ai/PriceEstimator.kt`
تستطيع تعديل الأسعار الأساسية لكل نوع حجر.

## تحسين الدقة لاحقاً
لجعل التطبيق يتعرف على الأحجار الكريمة بدقة عالية، ستحتاج:
- Dataset صور أحجار كريمة (مصنفة)
- تدريب نموذج (TensorFlow / PyTorch) وتصديره TFLite
- استبدال ML Kit Labeling بنموذجك الخاص داخل التطبيق
