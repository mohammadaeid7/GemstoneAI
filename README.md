# GemstoneAI - Android App (Kotlin + Jetpack Compose)

هذا مشروع أولي لتطبيق أندرويد باستخدام Kotlin وJetpack Compose. يحتوي المشروع على شاشة واحدة بسيطة باللغة العربية (مرحبًا وزر للاختبار).

الافتراضات التي استخدمت:
- package: com.gemstoneai.app
- minSdk: 21
- Kotlin + Jetpack Compose

ماذا بعد:
- يمكنني توسيع التطبيق بشاشات إضافية، تسجيل دخول، تخزين محلي (Room)، أو ربط API حسب طلبك.
- ملف GitHub Actions الموجود في المستودع (.github/workflows/build-apk.yml) سيبني APK debug تلقائيًا عند الد��ع إلى الفرع main أو master.
